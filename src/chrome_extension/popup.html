Upload Artifacts (VS Code → ChatGPT)
Artifacts are single files exposed from VS Code over localhost (e.g. bundle.md or a ZIP). Select one or more and upload into ChatGPT. Chat limit: 10 files per chat. Project limit: 20 files per project.
